# JavaScript对象

### Object对象

对象可以创建自有属性，还可以从为原型对象里面继承属性。

##### 创建对象的两种方式

- ```javascript
  var person = new Object();
  person.name = '孙悟空';
  person.age = 18;
  ```

- ```javascript
  var person = {
      name: "孙悟空",
      age: 18
  };
  ```

##### 对象属性的访问方式

- 对象.属性名

- 对象['属性名']

##### 栈和堆

JavaScript在运行时数据是保存在堆和栈中的。栈内存用来保存变量和基本类型，堆内存用来保存对象。变量指向一个对象，变量中实际保存的是对象在堆中的内存地址。
