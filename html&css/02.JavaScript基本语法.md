JavaScript严格区分大小写

### 2.注释

- 单行注释 `//注释内容`

- 多行注释 `/*注释内容*/`

### 3.标识符

- 标识符表示变量、函数、属性的名字

- 只能由字母、数字、下划线或者$组成，首个字母不能为数字

- 建议使用驼峰命名法

- 标识符不能是**关键字**或者**保留字**

- 关键字

| try        | catch  | finally | throw    | function | void  |
